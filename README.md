# OceanFleet
OceanFleet App is a console-based Java application developed using Object-Oriented Programming principles and GitFlow.

# Objective
To store and analyze vessel performance data and identify high-performance vessels.

# Use Cases
UC1: Create Vessel model

UC2: Store vessel records

UC3: Retrieve vessel by ID

UC4: Identify high-performance vessels

UC5: Console-based interaction

UC6: OOPS-based refactoring
